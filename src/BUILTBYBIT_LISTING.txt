[SIZE=5][B][COLOR=#3498DB]✦ Commands[/COLOR][/B][/SIZE]

[CODE]
PLAYER COMMANDS:
/duel <player>           - Challenge a player to a duel
/accept <player>         - Accept a duel request
/decline <player>        - Decline a duel request
/spectate <player>       - Spectate a player's match
/leave                   - Leave your current match/queue
/stats [player]          - View player statistics
/leaderboard [ladder]    - View ELO leaderboards
/elo [player]            - View ELO ratings
/kit                     - Open the kit editor
/rematch                 - Request/accept rematch
/ping [player]           - Check your ping

PARTY COMMANDS:
/party create            - Create a new party
/party invite <player>   - Invite a player
/party accept <player>   - Accept party invite
/party leave             - Leave your party
/party kick <player>     - Kick a member (leader only)
/party disband           - Disband the party (leader only)
/party info              - View party information

STAFF COMMANDS:
/mod                     - Toggle staff mod mode
/arena create <name>     - Create arena
/arena seta <name>       - Set spawn A
/arena setb <name>       - Set spawn B
/setspawn                - Set practice spawn
/seteditor               - Set kit editor location

QRANKS COMMANDS:
/rank <player>           - View player's rank
/grant <player>          - Open rank grant GUI
/grants <player>         - View grant history
/staff                   - View online staff
/arena                   - Arena management
/setspawn                - Set the spawn location
/seteditor               - Set the kit editor location
/elo <player> <ladder> <amount> - Modify player ELO
[/CODE]

[SIZE=5][B][COLOR=#3498DB]✦ Arena Setup[/COLOR][/B][/SIZE]

[CODE]
1. /arena create <name>      - Create new arena
2. /arena setspawna <name>   - Set spawn point A (stand where you want)
3. /arena setspawnb <name>   - Set spawn point B (stand where you want)
4. /arena setmin <name>      - Set arena boundary min corner
5. /arena setmax <name>      - Set arena boundary max corner
6. /arena enable <name>      - Enable the arena
7. /arena setsumo <name>     - Mark as sumo arena (optional)
[/CODE]
